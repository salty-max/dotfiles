The project uses `react-native-solar-icons/icons/linear` for all icons.

Import pattern: `import { ArrowRight, Shield } from 'react-native-solar-icons/icons/linear'`

Icon component type: `ComponentType<any>` (for props accepting icon components).

The icon API: `<IconName size={20} color="#FF0000" />`
